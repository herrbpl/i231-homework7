Write a method to create a Huffman prefix code for a given byte array (as constructor) and methods to code and decode data. Code must be optimal and decoding must completely restore the original array. Do not forget to test byte array of length 1. Have a look at: http://www.itcollege.ee/~jpoial/algoritmid/kodeerimisyl.html .

Koostada meetodid Huffmani prefikskoodi loomiseks etteantud kodeeritava baitmassiivi põhjal (konstruktorina), selle baitmassiivi kodeerimiseks ning kodeeritud massiivi dekodeerimiseks. Kontrollitakse koodi optimaalsust ning kodeeriva ja dekodeeriva meetodi kooskõlalisust. Kodeeritav massiiv võib sisaldada ka ühtainsat baiti (testige see juhtum eraldi). Vt. ka http://www.itcollege.ee/~jpoial/algoritmid/kodeerimisyl.html .
Kui kasutate mingeid allikaid, siis palun viidake nendele programmi tekstis.
